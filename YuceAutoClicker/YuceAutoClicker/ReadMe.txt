# Yuce Macro

Yuce Macro is a Windows application that simulates mouse clicks (left/right) when a specific key is pressed. Users can customize the click speed (CPS), click type (left/right), and which key triggers the macro. It is an ideal tool for gamers and for automating repetitive tasks.

## Features
- **Click Speed Adjustment (CPS):** Customize the click speed.
- **Click Type:** Simulate left or right mouse clicks.
- **Macro Key Selection:** Choose which key triggers the macro.
- **Simple and User-Friendly Interface:** Easy setup and usage.
- **Windows Platform:** Designed to work on Windows.

## Usage
1. Run the program.
2. Set the click speed and click type options.
3. Select the macro key.
4. Click the "Start" button to begin.
